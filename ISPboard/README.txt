#####################################
# README for ISPboard               #
# ###################################
# Project description:
# Board for attaching DIP28 Atmel chips
# and program and debug them over ISP.
# Also features 2 LEDs and 2 buttons that
# can be connected via jumpers to quickly test
# chip programming.
#####################################
# Revisions                         #
#####################################
Mk I
LEDS and buttons, connection for ISP.

Mk II
Added a PSU from battery and an on/off-switch.
Added pullups to buttons.
